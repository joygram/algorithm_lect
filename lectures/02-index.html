II. 선형 구조

5분씩 읽고 넘어가기 · 프로그래밍기능사 대비

← 전체 목차

샌즈가 있는 메뉴처럼 들어갔다 나오기(스택), 대사·이벤트 순서(큐). 한 편 약 5분씩이에요.

  • 1. 선형 구조가 뭐예요? (약 5분)→ 스택/큐/덱 차이. 메뉴·대사 흐름에 다 써요
  • 2. 큐(Queue) — 줄 서기 (약 5분)→ 대사·이벤트가 순서대로 나가게. FIFO, enqueue/dequeue
  • 3. 스택(Stack) — 접시 쌓기 (약 5분)→ 메뉴 뒤로가기·이전 화면이 스택. push/pop 순서 한 번 풀어보기
  • 4. 덱(Deque) — 앞뒤로 넣고 빼기 (약 5분)→ 앞뒤로 넣고 빼는 구조. 이름만 알아두면 돼요

실습·심화는 각 강의 페이지 안에 이어져 있어요. 4편(덱) 맨 아래에 II단원 심화 실습(4단계)이 삽입돼 있습니다. II단원 확장: 게임 시스템 실습과제 스택 손실습 큐 손실습 덱 손실습 기출 유형
