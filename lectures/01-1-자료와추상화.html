자료란 뭘까요?

자료(Data, 데이터)는 컴퓨터가 다루는 모든 정보예요. 숫자, 문자, 사진, 동영상, 프로그램 안의 변수값까지 다 자료입니다. 프로그래밍기능사 필기에서 "데이터 타입", "자료 추상화"가 자주 나와요.

데이터 타입

자료의 "종류"를 구분한 거예요.

  • 정수: 1, 2, 3, -5 같은 자연수·음수 (소수점 없음)
  • 실수: 1.5, 3.14 같은 소수점 있는 수
  • 문자: 'A', '가' 처럼 글자 하나
  • 문자열: "안녕", "Hello" 처럼 글자 여러 개
  • 논리형(불리언): 참(true, 트루) / 거짓(false, 폴스)

자료 추상화 · ADT

ADT = Abstract Data Type

한글: 추상 자료형  |  읽는 법: 에이디티(ADT), 앱스트랙트 데이터 타입(Abstract Data Type)

자료 추상화는 "어떤 걸 넣고, 빼고, 찾을 수 있다"처럼 동작만 정해 두고, "메모리 어디에 어떻게 저장하는지"는 숨기는 거예요. 시험에는 위처럼 ADT(추상 자료형)이라고 나옵니다.

예: 게임에서 "인벤토리"라는 ADT는 "넣기", "꺼내기", "순서 바꾸기"만 정하고, 안쪽을 배열로 할지 연결 리스트로 할지는 구현할 때 정해요.

🎮 이 단원이 게임에서 쓰이는 곳 (언더테일·RPG)

자료·추상화는 게임 기획서처럼 "인벤토리는 넣기/꺼내기만 한다", "대사는 순서대로 나간다"처럼 할 수 있는 연산만 정하는 단계예요. 실제로는 다음 단원부터 배우는 리스트·스택·큐로 구현합니다.

실습하기

자료구조를 코드로 다뤄 보려면 아래 iframe에서 리스트 예제를 바로 실행해 보세요. (다른 페이지로 넘어가지 않습니다)

📌 실습 전: 이 실습에서 쓰는 파이썬 API·함수

대응 파이썬 list는 순서 있는 배열처럼 씁니다. "맨 뒤에 넣기"·"몇 번째 칸 지우기"에 아래 메서드를 써요.

리스트.append(값)
맨 뒤에 한 칸 추가. 리스트 길이가 n이면 새 값은 인덱스 n에 들어갑니다.
리스트.pop(인덱스)
지정한 인덱스의 값을 제거하고 그 값을 반환해요. 예: pop(1)이면 1번 칸(두 번째 값)이 사라집니다.

왜 이렇게 쓰나요? 파이썬 list는 "동적 배열"이라 맨 뒤 추가는 빠르고, 인덱스로 접근·삭제가 직관적이에요.

👇 아래 코드가 실습창과 동일한 예제예요. 빈칸(_____)을 채운 뒤 실습창에서 그대로 실행해 보세요.

# 리스트: 맨 뒤에 추가 → append, 인덱스로 제거 → pop(인덱스)
arr = [10, 20, 30]
# ① 빈칸: 리스트 맨 뒤에 40을 추가하는 메서드 (append / insert / pop 중)
arr._____(40)
print("append(40) 후:", arr)
# ② 빈칸: 1번 인덱스(20)를 제거하려면? pop( ? )
arr.pop(_____)
print("pop(1) 후:", arr)
print("길이:", len(arr))
