파이썬으로 메뉴 뒤로가기·대사 큐 구현 (4단계)

← II단원 목차 전체 목차

스택과 큐를 파이썬으로 써서 메뉴 화면 히스토리, 대사 순서를 구현해 보는 심화 실습이에요.

🎮 이 심화에서 다루는 게임 활용

스택 = 타이틀 → 메인 → 설정 들어갈 때마다 push, "뒤로" 누르면 pop. 큐 = 대사 줄을 순서대로 enqueue, 화면에 띄울 때마다 dequeue.

심화 실습 4단계

  1. 1단계: 스택으로 메뉴 화면 이름 쌓기 — 리스트에 "타이틀", "메인", "설정"을 append로 넣고, pop으로 "뒤로" 할 때마다 꺼내 보기.
  2. 2단계: 큐로 대사 줄 넣고 꺼내기 — deque에 "안녕!", "선택지가 있어요", "1. 예 2. 아니오"를 넣고 popleft()로 순서대로 꺼내 보기.
  3. 3단계: 스택+큐 함께 쓰기 — 메뉴 스택과 대사 큐를 각각 변수로 두고, "메뉴 들어가기" / "대사 다음" 시뮬레이션을 print로 확인.
  4. 4단계: 간단한 메뉴 루프 흉내 — while과 input은 쓰지 않고, 미리 정해 둔 push/pop 순서를 코드로 써서 "들어가기·뒤로가기" 순서가 스택에 어떻게 쌓이는지 확인.
