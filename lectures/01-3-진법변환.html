3. 진법 변환 (10↔2↔16) · 약 5분

← 전체 목차 이 단원 목차

13을 2진수로 바꾸면? 언더테일처럼 세이브 파일·HP·데미지 숫자도 결국 0과 1로 저장돼요. 방법 네 가지만 보면 돼요.

진법 변환 — 네 가지만 알면 됨

① 10진수 → 2진수

2로 계속 나눠서 나머지만 적고, 나머지를 아래에서 위로 읽으면 끝.

예: 13 → 13÷2=6…1, 6÷2=3…0, 3÷2=1…1, 1÷2=0…1 → 나머지 1,0,1,1 거꾸로 → 1101₂

② 2진수 → 10진수

자리마다 2의 거듭제곱 곱해서 더하기. 오른쪽부터 2⁰, 2¹, 2², …

예: 1101₂ = 1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8+4+0+1 = 13

③ 10진수 → 16진수

16으로 계속 나눈 나머지 모으기. 10~15는 A~F.

예: 255 → 나머지 15(F), 15(F) → FF₁₆

④ 2진수 ↔ 16진수

2진수 네 자리 = 16진수 한 자리. 1111=F, 1010=A 같은 식.

예: 1111 1010₂ → FA₁₆  |  FA₁₆ → 11111010₂

문제 및 해설

10진수 25를 2진수로 바꾸면?
정답: 11001₂. 25÷2=12…1, 12÷2=6…0, 6÷2=3…0, 3÷2=1…1, 1÷2=0…1 → 나머지 1,0,0,1,1을 아래에서 위로 읽으면 11001.

문제 및 답 확인

아래 빈칸에 13을 2진수로 바꾼 과정을 채우고, 2진수 답을 입력한 뒤 [확인]을 누르세요.

13÷2=6 나머지 → 6÷2=3 나머지 → 3÷2=1 나머지 → 1÷2=0 나머지
