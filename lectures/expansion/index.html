게임 시스템 확장 실습과제

학습한 내용을 구체적인 게임 시스템으로 확장하는 추가 강의

← 전체 목차

아래 페이지들은 강의 외 추가 강의예요. 각 단원에서 배운 자료구조·알고리즘을 게임에서 실제로 쓰는 시스템(비트 플래그, 인벤토리, 메뉴 히스토리, 대사 큐, 인벤 정렬, 아이템/스킬 검색)으로 확장합니다. 각 주제마다 자료구조 복기 → 사용하는 파이썬 API 설명 → 실제 구현 활용 순으로 짚어요. 코드는 파이썬 기반입니다.

  • I단원 확장: 자료·진법·리스트 — 비트 플래그(업적/상태), 인벤토리(리스트), 세이브 데이터 묶기
  • II단원 확장: 스택·큐 — 메뉴 화면 히스토리(스택), 대사·이벤트 순서(큐)
  • III단원 확장: 트리·그래프 — 스토리 분기(트리), 맵·방 연결, DFS/BFS·최단 거리
  • IV단원 확장: 정렬 — 인벤토리 정렬(이름순·가격순), 버블·선택·삽입 정렬과 게임 활용
  • V단원 확장: 탐색 — 아이템/스킬 테이블에서 ID로 검색(이진 탐색)

진법은 단순 "이진수 표기"만이 아니라, 비트 플래그로 여러 상태를 한 숫자에 담는 구현, 2^n 슬롯 인덱스 등 자료구조·실제 활용과 연결된 부분을 각 확장에서 짚어요.
