01. 비선형 구조가 뭐예요?

자료가 한 줄이 아니라 가지처럼 퍼지거나, 서로 여러 개가 연결된 구조예요. 가족 관계도, 폴더 안에 폴더가 있는 구조, 지도 위의 길 연결처럼요.

02. 트리(Tree) — 나무처럼 가지치기

트리는 맨 위에서부터 아래로 뻗어 나가는 구조예요. 한 지점에서 여러 개가 갈라질 수 있어요.

  • 루트: 맨 위의 시작점 (나무 뿌리처럼)
  • 부모 / 자식: 위에 있는 게 부모, 바로 아래 연결된 게 자식
  • 형제: 같은 부모 아래의 자식들끼리
  • 리프(잎): 맨 끝, 자식이 없는 노드
  • 이진 트리: 각 노드의 자식이 최대 2개만 있는 트리 (왼쪽 자식, 오른쪽 자식)
📖 어원 Tree = 나무, Root = 뿌리. 위에서 가지가 뻗어 내려가는 모양이 나무 같아서 트리, 맨 위 시작점을 루트(뿌리)라고 해요. 더 보기 →

순회는 트리의 모든 노드를 어떤 순서로 방문할지 정한 거예요.

  • 전위: (나) → 왼쪽 → 오른쪽
  • 중위: 왼쪽 → (나) → 오른쪽
  • 후위: 왼쪽 → 오른쪽 → (나)

시험에 "이 트리를 전위 순회하면?" 같은 문제가 자주 나와요.

📋 전년도 기출 유형
루트 1, 왼쪽 2(자식 4,5), 오른쪽 3(자식 6,7)인 이진 트리에서 중위 순회(왼쪽 → 나 → 오른쪽) 결과는?
정답: 4, 2, 5, 1, 6, 3, 7. 더 많은 예제는 기출 유형 예제에서 풀어 보세요.

03. 그래프(Graph) — 점과 선으로 연결

그래프는 점(정점)과 선(간선)으로 "누가 누구와 연결되어 있는지" 나타내는 구조예요. 지하철 노선도, SNS 친구 관계처럼요.

  • 정점(vertex): 하나하나의 점 (역, 사람 등)
  • 간선(edge): 두 정점을 잇는 선 (길, 관계 등)
  • 방향이 있으면 "일방통로", 없으면 "왕복"이라고 생각하면 돼요.

탐색 방법

  • DFS(깊이 우선): 한 길로 쭉 들어갔다가, 막히면 돌아와서 다른 길. 스택을 쓰는 방법이에요.
  • BFS(너비 우선): 가까운 것부터 층층이. 큐를 쓰는 방법이에요. "최단 거리" 구할 때 자주 씁니다. 시험에도 나와요.
