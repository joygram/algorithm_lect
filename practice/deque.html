← 목차 II단원 4. 덱 (강의)

덱(Deque) = 앞·뒤 둘 다 넣고 뺄 수 있음. 큐처럼 쓰려면 뒤에 넣고 앞에서만 꺼내면 됨. 파이썬 collections.deque: append, appendleft, pop, popleft. 응용 최근 본 메뉴 N개, 슬라이딩 윈도우.

덱은 앞과 뒤 둘 다 넣고 뺄 수 있어요. 값을 입력한 뒤 앞에 넣기 또는 뒤에 넣기를 누르고, 꺼낼 때는 앞에서 꺼내기 / 뒤에서 꺼내기 중 하나를 골라 보세요.
